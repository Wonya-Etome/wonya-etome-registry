🔥 Step 1 — Create a free Firebase project

Firebase is Google's free database that syncs updates to all WhatsApp members instantly.

1
Go to console.firebase.google.com and click Add project
2
Name it wonya-etome · disable Google Analytics · click Create project
3
Click Build → Realtime Database → Create database · choose any location · start in test mode
4
Click the ⚙️ gear → Project settings → scroll to Your apps → click </> Web → register app → copy the firebaseConfig object

📋 Step 2 — Paste your Firebase config

Paste the entire const firebaseConfig = { … } object (or just the JSON inside the braces).

🔐 Step 3 — Set admin PIN

Choose a 4–6 digit PIN for authorized editors. You can add more admins later from the Admin panel. Share this PIN only with trusted members.

✏️ Edit Registry Directly

Enable edit mode to tap any payment status in the Registry or Category tabs to toggle it between Paid, Unpaid, and Pending. Changes sync to all users instantly.

👤 Add New Member

Add a member to the registry. They will appear in all views immediately.

📂 Add New Category

Add a new payment category (e.g. a new event or fund). All existing members will have their status for this category set to Pending.

🔑 Manage Admin PINs

Add PINs for other authorized editors. Each person gets their own PIN. PINs are hashed — they cannot be read, only verified.

🏷️ Edit Category Name / Year

Rename an existing category or change its year tag. All member data for this category is preserved.

✏️ Edit Member Name

Correct the spelling of an existing member's first or last name. All their payment data is preserved.

🗑️ Remove Member / Category

Select and permanently remove a member or category from the registry. This cannot be undone.
